# Pokémon FireRed and LeafGreen++

This is the repository for the Pokémon FireRed and LeafGreen++ hack. Currently, this hack simply adds the Physical/Special Split to [Deokishisu's  Pokemon FireRed and LeafGreen+ ROM hack](https://github.com/Deokishisu/FRLG-Plus).

Deokishisu's ROM hack serves to be the definitive way to experience FireRed/LeafGreen without departing too far from the vanilla experience. He provides many quality of life improvements from future games while adding several new Pokemon to the post-game in a way that seems natural.

One of the core philosophies of Deokishisu's ROM Hack is that the battle system of generation 3 should remain untouched. That means that a move's category--whether move is considered physical or special--is completely determined by the move's type. This leaves many Pokémon, like Gengar and Gyarados, unable to fully utilize their STAB moves, which rely on the lower of their Attack or Special Attack stats.

Pokemon FireRed and LeafGreen++ builds off of the amazing work done by Deokishisu, while providing the Physical/Special Split that many players have come accustomed to.

## How to play
1. [**Download the patch**](https://github.com/egalleta/FRLG-Plus/releases/latest)
2. Legally obtain a clean copy of Pokemon FireRed v1.0 (Sha1: 41cb23d8dccc8ebd7c649cd8fbb58eeace6e2fdc).
3. Apply the patch using an online patcher that is compatible with `.bps` files
4. Enjoy!

## Added Features
* Added Physical/Special Split.
    * The category of each move (i.e. Physical, Special, or Status) is based on the category of the moves in gen 4+.
* Added Physical, Special, and Status icons.
    * To see the category of a move, go to the move status screen and select a move to see the moves full summary. The move's category is added to the right of the move's power.

To see all the features added as part of Deokishisu's Hack, see the [**View the full features list for FireRed and LeafGreen+**](https://github.com/Deokishisu/FRLG-Plus/blob/master/FEATURES.md)
